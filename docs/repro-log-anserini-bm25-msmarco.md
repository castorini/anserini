# BM25 MS MARCO Passage Reproduction Log (Anserini)

## Environment
- OS: WSL2 Ubuntu 24.04
- Java: OpenJDK 21.0.10
- Anserini: 2.0.1-SNAPSHOT
- Hardware: Windows 10 laptop, 8GB heap allocated

## Steps Completed
1. Downloaded MS MARCO passage dataset (1009MB)
2. Converted collection to JSONL format (8,841,823 passages)
3. Built Lucene index using IndexCollection
4. Ran BM25 retrieval on dev-subset (6,980 queries)
5. Evaluated using trec_eval and msmarco_passage_eval

## Results
- MRR@10: 0.1840
- MAP: 0.1926
- NDCG@10: 0.2284
- Queries Ranked: 6,980

## Notes
- Used fatjar directly (appassembler not generated by Maven)
- Required -Xmx8g flag to avoid OutOfMemoryError
- TREC format output converted to MS MARCO format before MRR evaluation
