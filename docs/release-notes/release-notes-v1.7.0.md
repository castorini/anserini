# Anserini Release Notes (v1.7.0)

+ **Release date:** March 21, 2026
+ **Lucene version:** Lucene 9.9.1

## Summary of Changes

+ Refactored reproduction infrastructure, including renaming and reorganizing classes, configs, and CLIs.
+ Added CLIs for search, managing topics (queries), managing prebuilt indexes, and extracting queries and documents from a TREC run.
+ Added REST endpoint.
+ Added original BRIGHT topics.
