# Anserini Release Notes (v1.5.0)

+ **Release date:** January 9, 2026
+ **Lucene version:** Lucene 9.9.1

## Summary of Changes

+ Added symbol bindings for M-BEIR topics and qrels.
+ Added `FineWebCollection` and `SmolTalkCollection`.
+ Refactored fusion methods.
