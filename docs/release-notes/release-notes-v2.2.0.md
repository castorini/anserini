# Anserini Release Notes (v2.2.0)

+ **Release date:** June 7, 2026
+ **Lucene version:** Lucene 10.4.0

## Summary of Changes

+ Removed query-cached reproduction configs based on raw document collections.
+ Refactored from-document-collection reproduction summaries and refreshed reproduction docs catalogs.
+ Cleaned up and consolidated prebuilt-index reproduction configs and documentation.
+ Added BRIGHT and BEIR prebuilt reproduction docs.
+ Switched Korean language analysis to Lucene's `KoreanAnalyzer`, with updated Mr.TyDi and MIRACL effectiveness numbers.
+ Updated fatjar installation guidance and related agent skills.
