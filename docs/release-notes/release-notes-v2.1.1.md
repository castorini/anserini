# Anserini Release Notes (v2.1.1)

+ **Release date:** May 16, 2026
+ **Lucene version:** Lucene 10.4.0

## Summary of Changes

+ Fixed [`anserini#3269`](https://github.com/castorini/anserini/issues/3269).
